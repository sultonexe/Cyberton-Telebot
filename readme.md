## Cara Jalanin Botnya 
- Install NodeJs dan NPM , bisa dicek di google
- buka cmd/terminal lalu jalankan "node index" (tanpa tanda petik) 
- buka telegram cari username @CybertonBot , lalu start dan silahkan berinteraksi dengan bot
